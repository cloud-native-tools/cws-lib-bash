# Feature Detail: Development Environment

**Feature ID**: F013
**Name**: Development Environment
**Description**: IDE workspace setup and developer tool management.
**Status**: Implemented
**Created**: 2026-06-15
**Last Updated**: 2026-06-15

## Overview

Provides shell functions for configuring IDE workspaces (VS Code multi-root workspace management, language-specific search paths) and setting up developer tools (shell formatters, diagram generators, notebook environments). Streamlines development environment bootstrap across projects.

## Latest Review

Initial feature definition extracted from existing codebase scan. Covers `vscode.sh` (23 functions), `shfmt.sh` (4 functions), `plantuml.sh` (1 function), and `jupyter.sh` (1 function).

## Key Changes

1. VS Code workspace creation, folder management, and language search path configuration
2. Shell script formatting via shfmt (install, format, diff, check)
3. PlantUML server setup for diagram rendering
4. Jupyter notebook environment installation

## Implementation Notes

- `vscode.sh` manipulates `.code-workspace` JSON files via `jq`.
- Supports search path injection for Python, Rust, Go, and C/C++ projects.
- `shfmt.sh` handles binary installation and formatting operations.
- All tools are optional dependencies; functions check availability via `have`.
