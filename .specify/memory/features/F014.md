## Overview

Provides shell functions for common OpenSSL operations including Certificate Authority creation, CA-signed certificate generation, certificate inspection, and private key generation. Simplifies TLS/PKI setup for development and infrastructure environments.

## Latest Review

Initial feature definition extracted from existing codebase scan. Covers `openssl.sh` (4 functions).

## Key Changes

1. Self-signed CA creation (`openssl_create_ca`)
2. CA-signed certificate generation (`openssl_create_ca_signed_cert`)
3. Certificate viewing/inspection (`openssl_view_crt`)
4. Private key generation (`openssl_generate_private_key`)

## Implementation Notes

- Requires `openssl` binary in PATH.
- Functions accept all parameters (subject, key size, validity) as arguments.
- No hardcoded certificate subjects, paths, or organizational values.
