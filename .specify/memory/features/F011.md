## Overview

Provides shell functions for managing virtual machines using QEMU (cross-architecture emulation for ARM64 and AMD64) and libvirt (DHCP queries, network updates). Complements the virtualization helpers in `profile.d/09_virt.sh`.

## Latest Review

Initial feature definition extracted from existing codebase scan. Covers `qemu.sh` (3 functions) and `libvirt.sh` (2 functions).

## Key Changes

1. QEMU binary detection and cross-architecture VM launch (ARM64, AMD64)
2. Libvirt network management (DHCP leases, network configuration updates)

## Implementation Notes

- `qemu.sh` supports `qemu-system-aarch64` and `qemu-system-x86_64` binaries.
- `libvirt.sh` wraps `virsh` CLI commands.
- Cross-platform: QEMU available on both Linux and macOS; libvirt primarily Linux.
