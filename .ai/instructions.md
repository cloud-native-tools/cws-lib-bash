## Project Overview
CWS-Lib-Bash 是一个通用 Bash 函数库，按技术域拆分在 `scripts/*.sh`，并由 `profile.d/*.sh` 提供底层环境与公共能力。该库主要通过 `source ./bin/cws_bash_env` 注入当前 shell 会话，也可通过 `./bin/cws_bash_run` 以函数入口方式执行。

  - `test/`: `test_*.sh` 回归测试

## Runtime Loading Model (Critical)
- `bin/cws_bash_env` 会按顺序 `source profile.d/*.sh` 和 `scripts/*.sh`，因此库代码会在“加载期”执行。
- 加载期必须保持安静与轻量：避免无条件 stdout/stderr 输出、避免昂贵外部命令、避免副作用。
- 函数内使用 `return ${RETURN_FAILURE:-1}`，不要在库脚本里直接 `exit` 终止调用者 shell。
- 参数校验与依赖检查前置（优先 `have <cmd>`），错误通过 `log` 体系输出。
- 继续遵循 snake_case、`local` 变量、`${var}` 花括号引用、`${var:-default}` 默认值风格。

## Build/Test/Validation Commands
- 加载环境：`source ./bin/cws_bash_env`
- 运行函数：`./bin/cws_bash_run <function_name> [args...]`
- 执行全部测试：`./bin/cws_bash_test`
- 执行单测文件：`./bin/cws_bash_test test_env`
- Lint（全量）：`find "$PWD" -type f -name "*.sh" | xargs shellcheck -x`
